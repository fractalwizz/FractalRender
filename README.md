## FractalRender + PaletteMaker
Fractal Image Renderer written in Perl<br>
Compatible Palette Maker written in Perl<br>
V0.0/V0.0

### Module Dependencies
GD<br>
Math::Complex<br>

### PaletteMaker Usage
